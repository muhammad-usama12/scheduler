## Setup

Install dependencies with `npm install`

## Installing Missing Dependencies

Install dependencies: `npm i`

Missing dependencies can be installed with: `npm install 'package'`

## Dependencies

- ReactJS
- React Testing-Library
- Axios
- Babel
- Jest
- Storybook
- Sass
